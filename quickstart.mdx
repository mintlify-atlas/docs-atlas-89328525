---
title: 'Quickstart'
description: 'Get Mi Colección de Juegos running locally in a few minutes.'
---

Mi Colección de Juegos is a PHP/SQLite web app with no build step. You only need a PHP web server and a browser.

<Steps>
  <Step title="Check prerequisites">
    Confirm PHP 7.4+ is installed and the SQLite3 and PDO extensions are enabled.

    ```bash
    php -v
    php -m | grep -E 'sqlite|pdo'
    ```

    You should see `sqlite3`, `pdo_sqlite`, and `PDO` in the output. See [Requirements](/setup/requirements) for details.
  </Step>

  <Step title="Get the source code">
    Clone the repository or download and extract the archive.

    ```bash
    git clone https://github.com/your-org/mi-coleccion-de-juegos.git
    cd mi-coleccion-de-juegos
    ```

    There is no `composer install`, `npm install`, or build command to run.
  </Step>

  <Step title="Place files in your web server root">
    Copy or move the project folder into your web server's document root.

    <Tabs>
      <Tab title="XAMPP (Windows)">
        Copy the project folder into `C:\xampp\htdocs\`.

        ```
        C:\xampp\htdocs\mi-coleccion-de-juegos\
        ```

        <Note>
          XAMPP is the easiest option on Windows. Download it from [apachefriends.org](https://www.apachefriends.org) if you don't have it. Start both the **Apache** and nothing else — SQLite is file-based and requires no separate database server.
        </Note>
      </Tab>
      <Tab title="PHP built-in server">
        You can serve the project directly from its directory using PHP's built-in server — no Apache or XAMPP required.

        ```bash
        php -S localhost:8000
        ```

        Then open [http://localhost:8000](http://localhost:8000) in your browser.

        <Warning>
          The PHP built-in server is suitable for local development only. Do not use it in production.
        </Warning>
      </Tab>
      <Tab title="Apache / WAMP">
        Copy the project folder into your Apache document root (commonly `/var/www/html/` on Linux or `C:\wamp64\www\` on Windows with WAMP).

        ```
        /var/www/html/mi-coleccion-de-juegos/
        ```

        Make sure `mod_php` is enabled and the `php/` directory is writable by the web server user.
      </Tab>
    </Tabs>
  </Step>

  <Step title="Open the app in your browser">
    Navigate to the app in your browser. The exact URL depends on your setup:

    | Server | URL |
    |--------|-----|
    | XAMPP / WAMP | `http://localhost/mi-coleccion-de-juegos/` |
    | PHP built-in server | `http://localhost:8000/` |
    | Apache (Linux) | `http://localhost/mi-coleccion-de-juegos/` |

    The app opens `index.html` by default.
  </Step>

  <Step title="Verify the database initializes">
    On the first request to any PHP endpoint, `getDB()` in `php/setup_db.php` automatically creates the SQLite database file at `php/games.db` and sets up the `juegos` table.

    No manual SQL or migration commands are needed. If you see the main page without errors, the database is ready.

    <Tip>
      If you get a "permission denied" or "unable to open database" error, make the `php/` directory writable by your web server process. See [Database setup](/setup/database) for details.
    </Tip>
  </Step>
</Steps>
