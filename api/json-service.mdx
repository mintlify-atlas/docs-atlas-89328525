---
title: "json_service.php"
description: "JSON utility service for processing game ID requests."
---

Accepts a JSON body containing a game ID, validates the input, and returns a processing confirmation with a server-generated timestamp.

## POST — process a game ID

**Content-Type:** `application/json`

### Request body

<ParamField body="gameId" type="integer" required>
  The ID of the game to process. Must be present in the request body.
</ParamField>

### Validation errors

The server returns HTTP `400` if `gameId` is absent from the request body:

```json 400
{ "success": false, "message": "Falta el campo requerido: 'gameId'." }
```

### Response fields

<ResponseField name="success" type="boolean" required>
  `true` when the request was processed successfully.
</ResponseField>

<ResponseField name="message" type="string" required>
  Confirmation message including the received game ID.
</ResponseField>

<ResponseField name="timestamp" type="string" required>
  ISO 8601 timestamp of when the server processed the request.
</ResponseField>

### Success response

```json 200
{
  "success": true,
  "message": "Procesamiento JSON exitoso. Recibimos el ID: 123",
  "timestamp": "2026-03-22T10:00:00+00:00"
}
```

### Example

```bash cURL
curl --request POST \
  --url http://localhost/php/json_service.php \
  --header 'Content-Type: application/json' \
  --data '{"gameId": 123}'
```
