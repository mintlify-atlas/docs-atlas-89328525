# Mi Colección de Juegos

**Mi Colección de Juegos** is a full-stack web application that lets you catalog, search, and manage your personal video game library. Built with HTML, CSS, JavaScript, and a PHP/SQLite backend, it runs entirely in a standard web hosting environment with no external dependencies beyond a PHP server.

## How it works

The application is a single-page-style multi-HTML app served by a PHP web server. The frontend communicates with backend PHP services over `fetch`, sending and receiving JSON. All game data is stored in a local SQLite database file (`php/games.db`).

```
Browser (HTML + JS)
       │
       ├──► php/db_service.php       — list / search games
       ├──► php/form_service.php     — add a new game
       ├──► php/edit_service.php     — update a game
       ├──► php/delete_service.php   — remove a game
       ├──► php/json_service.php     — JSON utility service
       └──► php/third_party_service.php — FreeToGame recommendations
```
